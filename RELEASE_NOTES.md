# リリースノート

## バージョン 1.0.0 - 2025-12-12

### 🎉 新機能

#### 治療スケジュール管理
- **日付変更機能**: 治療スケジュールの日付を柔軟に変更できるようになりました
- **スケジュール一括リセット**: すべての治療予定を一度にクリアできる機能を追加
- **ルールベース自動配置**: 治療計画を自動的にスケジュールに配置する機能とカスタマイズオプション

#### 歯式チップ操作
- **まとめノードの分離・合体**: 歯式チップをドラッグ&ドロップで治療ノードを分離・合体できる機能を実装
- **空欄ドロップ分離**: 歯式チップを使った直感的な分離操作が可能に
- **配置済みノード操作**: 既に配置された治療ノードの歯式を柔軟に編集可能

#### ステップマスター機能
- **ステップマスター管理**: 治療ステップを一元管理できるマスター機能を実装
- **削除ステップの空欄表示**: ステップマスターから削除されたステップは空欄として表示されるように改善
- **step00の非表示化**: UI上でstep00をステップマスター一覧から非表示に

#### マスター管理機能
- **設定画面の全画面化**: マスター編集がより使いやすい全画面UIに刷新
- **マスタ編集機能**: 病名マスター・治療法マスターの編集機能を実装
- **色分け機能**: 病名マスターと治療法マスターに視覚的な色分けを追加

#### 病名管理
- **病名優先モード**: 歯をクリックするだけで即座に病名を適用できるモードを追加
- **排他的病名ルール**: 特定の病名が選択された際に他の病名を自動的に除外する機能
  - 複数病名への対応拡張
  - グループベースのルール適用
- **設定済み病名リスト**: 折りたたみ機能付きの病名リスト表示

### 🐛 バグ修正

- **歯式チップ分離時のバグ**: 元ノードから歯式が削除されない問題を修正
- **治療最終日偏り**: ルールベース自動配置で治療が最終日に偏ってしまう問題を修正
- **治療内容変更時のリセット**: 治療内容を変更した際にスケジュール配置がリセットされる問題を修正
- **ルールベース設定の永続化**: ルールベース設定がローカルストレージに正しく保存されるように修正

### ♻️ リファクタリング・UI改善

- **治療ノードUIの全面刷新**: より見やすく使いやすいデザインに一新
- **すべてクリアボタンの配置**: 常に固定位置に表示され、治療ノード設定の直下に配置
- **病名優先モードUI**: 歯式の下に移動し、より直感的な操作性を実現
- **空欄ドロップゾーンの削除**: ノード外ドロップで同じ場所に分離できるように改善
- **応答性の改善**: 治療内容変更時のパフォーマンスを最適化

### 📝 技術的な改善

- ドラッグ&ドロップ操作の全体的な改善
- ローカルストレージによる設定の永続化
- React Flowを活用した治療フローの可視化
- 状態管理の最適化によるパフォーマンス向上

## 初回リリース

このバージョンは、歯科治療フローを視覚的に管理できる革新的なアプリケーションの初回リリースです。治療計画の立案からスケジューリング、病名・歯式の管理まで、歯科治療に必要な機能を包括的にサポートします。

### 主な特徴

- 🦷 直感的な歯式選択UI
- 📅 柔軟な治療スケジュール管理
- 🔄 ドラッグ&ドロップによる治療ノード操作
- 📋 マスターデータの一元管理
- 🎨 視覚的に分かりやすいカラーコーディング
- ⚡ 自動配置機能による効率的な計画立案
